## MusicFunApp

Modern music streaming web application built with React, TypeScript, Redux Toolkit, and RTK Query.  
Features playlist management, API caching, mutations, and optimized data fetching.

---

## Tech Stack

- React
- TypeScript
- Redux Toolkit
- RTK Query
- Vite
- CSS / SCSS

---

## Features

- Create and manage playlists
- Fetch music via API
- Optimized caching
- Lazy queries
- Error & loading states
